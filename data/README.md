# Data

Five CSVs make up a small star-schema dataset: `Customers`, `Products`, `Orders`, `Order_Items`, `Returns`. All are generated for practice/portfolio use, no real customers, no PII, full-size (largest file is 76 KB), and included as-is since there's nothing large or sensitive to sample down.

See [`docs/data_dictionary.md`](../docs/data_dictionary.md) for column-level definitions, and [`sql/00_schema.sql`](../sql/00_schema.sql) for the loadable DDL.
